# 🌍 CareAid - Charity Donation Management System

A web-based application designed to simplify and manage charity donations efficiently.  
It allows donors to contribute to campaigns, track donations, and view reports in real time.

---

## 📌 Features

### 👤 Donor Features
- User login system
- Browse and donate to campaigns
- View donation history
- Download receipts (extendable feature)
- Real-time donation tracking

### 🧑‍💼 Admin Features
- Manage donation campaigns
- View total donations and donor details
- Generate donor reports
- Monitor campaign progress

### 📊 Dashboard
- Total donations counter
- Total donors count
- Recent donation display
- Live donation chart (Chart.js)

### 🔐 Security Features
- Session-based login (basic)
- Local storage data handling (can be upgraded to database)
- Input validation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Tailwind CSS
- Chart.js
- LocalStorage (for data storage)
